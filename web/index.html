News aggregator

Examples:
URL: https://www.netroby.com/rssRULE: Title:NewTitle
URL: http://feeds.nytimes.com/nyt/rss/TechnologyRULE: Title=title_field,Description=Body
Rule pattern is 'name_of_field_to_extract=new_field_name'. Pairs separator - ','. If field's name dont change, 'new_field_name' may be omitted.
WARNING: field names are case-sensitive.
Rule example: 'Title,Description:Body,Published:Added'
